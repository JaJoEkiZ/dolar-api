# Funcionalidad:
- Calcular el valor del dolar en Pesos Argentinos con las cotizaciones obtenidas desde [Dolar-Api](https://dolarapi.com).

# Datos de input:
 - Monto en dólares.
 - Tipo de dólar.

# Respuesta obtenida:
  - Tipo de Dólar.
  - Valor a cotizar.
  - Cotización actual.
  - Valor en Pesos Argentinos.
